 # Desigen：论文复现项目

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![PyTorch](https://img.shields.io/badge/PyTorch-1.9+-ee4c2c.svg)](https://pytorch.org/)
[![Jittor](https://img.shields.io/badge/Jittor-1.3+-blue.svg)](https://jittor.org/)


本项目是对论文《 Desigen: A Pipeline for Controllable Design Template Generation》的完整复现实现，同时提供了 **PyTorch** 和 **Jittor** 两个深度学习框架的版本。
